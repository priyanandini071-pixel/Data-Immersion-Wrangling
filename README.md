# Data Immersion & Wrangling

## Objective
Acquire, clean, transform, and prepare data for analysis.

## Dataset
Sales Dataset with customer, product and sales information.

## Data Quality Issues Found
- Missing values in Age
- Missing values in City

## Cleaning Performed
- Filled missing Age with median
- Filled missing City with Unknown
- Standardized date formats
- Removed duplicate rows

## Feature Engineering
- Order_Month
- Sales_Per_Unit

- Streamlit
- Matplotlib
